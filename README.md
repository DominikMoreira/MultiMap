# MultiMap

MultiMap is a SwiftUI sample app for exploring interactive maps with Apple Maps.

**What is this project about?**
- Searching for places and showing them on a map
- Selecting multiple places from a sidebar list
- Quickly deleting or focusing locations
- Learning how to make SwiftUI apps that use MapKit and modern UI features

**Key SwiftUI techniques covered:**
- Context menus for extra actions (like deleting items)
- Saving user info with @AppStorage
- Responding to keyboard shortcuts
- Multi-selection in lists
- Automatically reacting to changes in selection
- Searching for real-world locations using MKLocalSearch
- Identifying data with UUID and custom Hashable

This project is ideal for anyone wanting to learn practical mapping features and advanced SwiftUI patterns. Try searching for a city, select locations, and see them instantly appear on the map.
